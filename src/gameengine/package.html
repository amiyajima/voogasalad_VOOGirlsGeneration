VOOGirlsGeneration - Game Engine
Package Summary:
This packages provides the classes and interfaces required to instantiate and run a Game containing all components created in the Authoring Environment as contained within the JSON.
Package Contents:
public class GameEngine - Initialize a GameLoop, JSONParser and prepares the engine for game loading
public class Player - A player object that contains the logic for playing each level. This object requires no parameters for initialization.
public class GameLoop - Main GameLoop of the Game Engine.
