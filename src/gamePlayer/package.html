VOOGirlsGeneration - Game Player
Package Summary:
This packages provides the classes and interfaces required to play a Game that has been developed through the Game Authoring Environment utilizing the Game Engine. The Player accepts user import accordingly and displays the current state of the game to the player.
InitialScene View - Responsible for Loading and Saving the game being played.
Game Space View - Responsible for Playing the game and associated functions.
Score Board View - Responsible for showing the score of the current game being played.
