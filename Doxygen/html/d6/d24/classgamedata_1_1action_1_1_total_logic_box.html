Detailed Description

GUI element used in ActionEditor. This allows a user to specify a whole stat modifying operation. In a stats-modifying action, each operation adds to the target stat the value from a reference, scaled by a positive or negative number. This GUI element allows the user to which stat to edit and who the stat belongs to (actor or receiver).
